# Androxy
Android Global HTTP Proxy switcher

## Setup
1. Install app
2. Exec adb command  
```adb shell pm grant com.android.example.androxy android.permission.WRITE_SECURE_SETTINGS```  
(Androxy requires android.permission.WRITE_SECURE_SETTINGS permission)
